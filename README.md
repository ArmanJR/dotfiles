# dotfiles

Arman's personal configs for terminals, shells, and apps.

## Platforms

- **`mac/`** — macOS (Apple Silicon) with Homebrew, Zed, VSCode, Ghostty, Claude Code
- **`linux-edge/`** — Linux edge devices (Raspberry Pi, Nvidia Jetson, ARM/x86) with minimal tooling
- **`linux-server/`** — Linux servers and VMs with cloud tools (AWS/GCP/Terraform), AI tools, Claude Code
